## Installation

Create `.env` file in the root directory and add environment variables in it:

```sh
BOT_TOKEN=
MONGO_USERNAME=
MONGO_PASSWORD=
```

Install the dependencies and devDependencies and start the project:

```sh
npm i
npm run build
npm start
```

For development environment use:

```sh
npm i
npm run dev
```
